# Phyllotaxis
**Status:** Live
**Last Updated:** 2026-03-03
**Location in site:** `/lab/phyllotaxis`

## What it is
Animated phyllotaxis pattern showing plant seed packing from the golden angle (~137.5°) and its relation to Fibonacci spirals. 30-second autonomous demo cycle: seed growth, zoom out, angle morphing, and spiral arm highlighting.

## How to run / develop
Canvas 2D visualiser. Component at `core/PhyllotaxisVisualiser.tsx`. No dependencies beyond React.

## Current state
Demo mode complete. 1,200 seeds on desktop, 600 on mobile. Shows golden angle packing, angle comparison, and 13/21 Fibonacci spiral families.
